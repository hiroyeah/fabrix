各.pyと使い方について

STLファイルをインポートしてAAMを行う場合

1．
meshtoNERB コマンド で変換を行い、
メッシュを削除し、ポリサーフェスを使う。
ポリサーフェスを　_Join （結合)コマンドで結合する

2. 
makeSliceSurface_cmd.py
->曲面からAAMできる面だけを残すスクリプト
コマンドを入力すると、角度について問われるので、
斜面の角度を入力　（AIBOではホットエンドの角度から３０度と入力をしている）

3.
AAM_CurvedSurface_cmd.py
->曲面上にAAMを行うスクリプト
付け足すもの　と　付け足すベースを選択すると、スライスが開始される。




その他の
__plugin__.py
AAM_cmd.py
AAM_Rhino.py
gcodeBuilder.py

は現時点では必要なく、
過去の作成したスクリプトの参照として残しておきます
